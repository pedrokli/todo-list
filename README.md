# todo-list
Esse projeto é sobre uma aplicação de lista de tarefas em Java, onde é possível cadastrar tareafs, listar tarefas, buscar tarefas pelo id, editar e excluir tarefas(CRUD)

## Teconologias utilizadas:
- Java 17
- Spring
- H2 Databases
- Swagger(Springdoc)
- Fly.io
- Postresql
